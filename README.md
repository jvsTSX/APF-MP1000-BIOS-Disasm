# APF MP1000 BIOS Disassembly
disassembly of the APF MP1000 console's BIOS ROM

assemble with WLA-DX 
```
wla-6800 -o apf_4000_commented.o apf_4000_commented.asm
wlalink lfbios apf_4000_asemd.bin
```
